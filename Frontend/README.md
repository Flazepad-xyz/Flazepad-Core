# LoadingUpCoin Frontend UI

LoadingUpCoin is a decentralized application (dApp) frontend built with React, leveraging Web3 technologies to provide seamless blockchain interactions. This project integrates Web3Modal, Wagmi, and other tools to enable users to connect to blockchain networks like Ethereum Testnet(BSC).

## Features

- **Web3 Integration**: Connect to blockchain networks using Web3Modal.
- **Customizable Theme**: Easily modify the UI theme using `themeVariables`.
- **Blockchain Support**: Currently supports Ethereum Testnet(BSC).
- **State Management**: Uses `QueryClient` for efficient state and data management.
- **Responsive Design**: Built with Tailwind CSS for a modern and responsive UI.

## Project Structure

- **`src/`**: Contains the main application code.
- **`public/`**: Static assets like images, fonts, and the main `index.html`.
- **`build/`**: Compiled production-ready files.
- **`tailwind.config.js`**: Tailwind CSS configuration for styling.
